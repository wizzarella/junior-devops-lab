# Terraform

This directory contains a first Terraform configuration for learning purposes.

It uses the `local` provider to create a local text file.  
No cloud resources are created.

## Commands

Initialize Terraform:

```bash
terraform init
```

Format the configuration:

```bash
terraform fmt
```

Validate the configuration:

```bash
terraform validate
```

Preview changes:

```bash
terraform plan
```

Apply changes:

```bash
terraform apply
```
