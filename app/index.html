Junior DevOps Lab

A hands-on project for learning Docker, Docker Compose, GitHub Actions, Kubernetes manifests, and Terraform basics by building a small static site delivery workflow.

What This Lab Shows

Containerized Web App

A static HTML page served by Nginx inside a Docker container.

Repeatable Local Run

Docker Compose stores the local run configuration in version control.

CI Validation

GitHub Actions checks Docker and Terraform changes before merging.

Infrastructure Basics

Kubernetes and Terraform examples introduce deployment and IaC concepts.
